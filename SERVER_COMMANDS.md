## Quick Patterns

### Check the server

```bash
bp-start
```

### Check incoming work

```bash
bp-intake
bp-batches
bp-files
```

### See the server/shared folder layout

```bash
bp-tree
```

### Process a new drop

```bash
bp-incoming
bp-process phone-drop
bp-latest
bp-files
```

### Check backups

```bash
bp-backups
```
