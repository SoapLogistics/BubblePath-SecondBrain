# BubblePathPhone

BubblePathPhone is a SwiftUI iPhone-first app scaffold for a calm, living thought space.

It opens onto a floating bubble canvas where tapping empty space creates a new bubble. Tapping a bubble opens a writing detail view with a title, body, created date, and last edited date. Bubbles persist locally on disk.

## Build

From this folder:

```bash
swift build
```

The code is organized into app entry, models, persistence, haptics, and views.
